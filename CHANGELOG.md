# Changelog

## [1.1.0] - 2025-12-29
### Added
- Added Russian language support with localized stop words.
- Implemented Unicode-safe tokenization and lowercase handling for Cyrillic support.
- Added language selection setting to the plugin dashboard.

## [1.0.1] - 2025-12-28
### Fixed
- Fixed auto-generation logic to work with "Draft" and "Pending" post statuses.
- Corrected JS asset pathing for the manual generation button.
- Improved default settings handling for new installations.

## [1.0.0] - 2025-12-28
### Added
- Core Luhn Summarization Engine with cluster scoring logic.
- Admin Settings page under "Settings > Luhn Excerpts".
- Auto-generation hook for new posts.
- Manual "Generate" button for Gutenberg and Classic Editor.
* Public GitHub Update Manager integration.
- PSR-4 Autoloader and strict typing (PHP 8.2).
